# QuickCart - C# Desktop Application

QuickCart is a C# Desktop Application developed using Windows Forms.

## Features
- Seller Dashboard
- Product Management
- Cart System
- Billing System

## Technologies Used
- C#
- .NET Framework
- Windows Forms
- Visual Studio

## How to Run
1. Clone the repository
2. Open QuickCart.sln in Visual Studio
3. Press *Start* to run
