# QuickCart - C# Desktop Application

QuickCart is a C# Desktop Application developed using Windows Forms.

## Features
- Seller Dashboard
- Product Management
- Cart System
- Billing System

## Technologies Used
- C#
- .NET Framework
- Windows Forms
- Visual Studio

## How to Run

### Using Visual Studio
1. Clone the repository or download ZIP
2. Open QuickCart.sln in Visual Studio
3. Set QuickCart as Startup Project
4. Press *Start (F5)* to run

### Requirements
- Visual Studio 2019 or later
- .NET Desktop Development workload
